# Bookmarks
___
## Приложенеие "Закладки" дает возможность делится картинками 
___
## Стек технологий

* **Python 3.11.7**
* **Django 5.0.2**
* **social-auth-app-django 5.4.0**
* **SQLite**
* **Redis**
___
## Функционал
* **Регистрация и авторизация пользователя**
* **Регистрация пользователей через Google, Facebook, Twitter**
* **Добавление картинок на доску при помощи закладки**
* **Отображение просмотров и лайков при помощи Redis**
* **Возможность подписаться на других пользователей**
___
## Установка
1. **Скачайте исходники c [GitHub](https://github.com/Refus1on/Todolist)**
2. **Заполните файл .env в соответствии с шаблоном .env.example**
3. **Запустите Redis `docker run -it --rm --name redis -p 6379:6379 redis`**
4. **Запустите сервер командой `python manage.py runserver`**
